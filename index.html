Set Up Cloud Sync

To sync your writing streak across all your devices, you'll need a free Firebase project. It takes about 3 minutes.

1 Go to Firebase Console and create a new project (free tier).
2 In your project, go to Build → Realtime Database and click Create Database. Choose any location, then select "Start in test mode".
3 Go to Project Settings (gear icon) → General → scroll to "Your apps" → click the Web (</>) icon. Register an app (any name), then copy the firebaseConfig object.
